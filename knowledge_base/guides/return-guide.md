# 退换货指南

## 如何退货

1. 登录您的账户，进入「我的订单」
2. 选择需要退货的订单，点击「申请退货」
3. 选择退货原因
4. 选择退货方式: 自行寄回或上门取件
5. 打印退货标签（或在快递点出示二维码）
6. 将商品连同配件和原包装妥善打包
7. 投递包裹或等待上门取件

## 退货条件

- 商品需保持原始状态，标签、配件和包装齐全
- 电子产品不得激活软件许可或撕除保护膜
- 服装鞋类需未经穿着，吊牌完好
- 化妆品和个护产品需未拆封，密封完好
- 数字商品、礼品卡和个性化定制商品不支持退货

## 退款时效

- 退货签收后: 1-2个工作日内发起退款
- 信用卡退款: 3-5个工作日到账
- 数字钱包退款: 1-2个工作日到账
- 店铺余额: 处理完毕后即时到账

## 换货流程

换货以退货加新订单的方式处理，确保您能最快收到替换商品。如需换货，先发起原商品退货，然后重新下单购买替换商品。如果您需要在退货处理前就收到替换商品，直接下单新商品即可。
